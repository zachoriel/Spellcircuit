** Thank you for purchasing the Awesome Effects - Complete Bundle! **

REQUIREMENTS:
-Unity 2017.10f3 or higher
-Post Processing Stack asset by Unity

HOWTO:
Install Post Processing Stack BEFORE installing this pack.
Drag AwesomeParticlesController.prefab from Mandatory Prefabs folder to your scene
and then drag any of the effect prefabs to the scene.

-To control colors, just use the gradient editor found on every prefab.
-Some prefabs have targets, the controls can be found on each prefab's inspector too.

Enjoy!
